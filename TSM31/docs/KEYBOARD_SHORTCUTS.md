## General Navigation

- **Tab**: Move between input fields
- **Shift+Tab**: Move backward between input fields
- **Enter**: Activate selected button
- **Space**: Toggle checkbox/switch
- **Arrow Keys**: Navigate dropdown selections

## Tips

- Press **ESC** at any time to return to the main menu
- The menu on the left shows available keys for your current context
- Selected tabs are highlighted in blue
- Test Power button shows current state (ON/OFF)
